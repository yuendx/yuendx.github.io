安装

  1. Git
  2. Node.js
  3. 备份本地博客目录的部分文件夹(如下)
  • _config.yml
  • package.json
  • .gitignore

配置 Git

  • 桌面右键选择bash here,设置用户名称和邮件地址
  • 完成后接着配置SSH

配置SSH Key

  • 然后连续按三次回车，会生成id_rsa.pub文件，用记事本打开它，然后全部复制即可，待会要用。这个文件的地址：C:\Users\Administrator.ssh (文件夹下)

  • 打开你的GitHub主页，打开设置->Deploy keys->Add deploy key.

  • 这里名称随便取，在下面粘贴内容保存即可。
